# AI-Based Mental Health Analyzer Using Social Media Text Data

## Description
This project analyzes social media text using Artificial Intelligence and Natural Language Processing to detect emotions and predict mental health risk levels.

## Technologies Used
- Python
- Flask
- HTML, CSS
- Hugging Face Transformers (BERT)
- NLP

## Features
- Emotion Detection
- Mental Health Risk Prediction
- AI Chatbot Support
- Web Dashboard

## How to Run
1. Install Python
2. Install libraries: pip install -r requirements.txt
3. Run: python app.py
4. Open browser: http://127.0.0.1:5000/

## Disclaimer
This system is not a medical diagnosis tool.
